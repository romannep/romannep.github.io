Разработки

  • Kate
  • Набор библиотек с конечной целью создания фреймворка для разработки бизнес ПО (учетных систем, CRM, PM и подобных)
  • kate-form - простое управление сложными интерфейсами на React/Redux
  • kate-form-material-kit-react - набор визуальных элементов для kate-form
  • kate-client - библиотека для разработки интерфейсов с императивным подходом к описанию поведеня форм
  • G-ITSM - демо использования kate-client для разработки таск-менеджера с настраиваемым жизненным циклом задач
  • katejs - фреймворк разаботки бизнес приложений
  • Разное
  • Magnet-strike - логическая игра - стройка здания с проверкой на прочность. Сделано на Phaser CE и Cordova для apk.
