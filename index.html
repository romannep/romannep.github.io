Разработки

  • Kate
  • Набор библиотек с конечной целью создания платформы для разработки организационного ПО (учетных систем, CRM, PM и подобных)
  • kate-form - простое управление сложными интерфейсами на React/Redux
  • kate-form-material-kit-react - набор визуальных элементов для kate-form
  • kate-form-demo - демо использования kate-form как отдельно, так и вместе с kate-form-material-kit-react
  • kate-client - (в разработке) фреймворк для разработки интерфейса организационного ПО
  • G-ITSM - демо использования kate-client для разработки таск-менеджера с настраиваемым жизненным циклом задач
